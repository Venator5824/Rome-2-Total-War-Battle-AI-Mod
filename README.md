# Rome-2-Total-War-Battle-AI-Mod
An LUA open-source mod that aims to overhaul the Rome 2 Total War AI by using native files and overtaking them with new functions, as well as adding new lua documents. 

Code: LUA
Tokens: Around 100 thousand so far
Newly created files: 9

Rome 2 Total War API list: https://bukowa.github.io/twr2docs/lua/api/interface/
Total War Battle Script Documentations: https://wiki.totalwar.com/w/Battle_Script_Documentation
and similar.

This mod is required and called by the main file: lib_script_ai_planner.lua
